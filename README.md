## Credit Card Application

A basic project of Credit card application that has the below functionalities
   displayInfo()-To display the Account Holder name,Account number,Address,Mobile number
   creditAmount()-Enables the Account holders to credit the Amount,so that the balance gets increased
   cashWithdraw()-Enables the Account holders to withdraw the amount,so that balance gets decreased
